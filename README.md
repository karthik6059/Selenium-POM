# 🚀 HRM Automation Framework

A robust **UI Automation Framework** built using **Java, Selenium, TestNG, and Maven** for testing HRM (Human Resource Management) applications.

---

## 🧰 Tech Stack

* 🧪 **Automation Tool:** Selenium WebDriver
* 🧩 **Test Framework:** TestNG
* 📦 **Build Tool:** Apache Maven
* 📊 **Reporting:** Extent Reports
* 📝 **Logging:** Log4j
* 📂 **Test Data:** Apache POI (Excel)
* 🗄️ **Database:** MySQL
* ☕ **Language:** Java (JDK 17)

---


## ⚙️ Key Features

✔ Page Object Model (POM) design
✔ Cross-browser support
✔ Headless execution support (CI ready)
✔ Extent Reports for detailed reporting
✔ Log4j logging integration
✔ Excel-based test data handling
✔ Database connectivity (MySQL)
✔ Maven-based execution

---
## 🔧 Dependencies Used

* Selenium `4.27.0`

---

## 🧠 Best Practices Followed

* Separation of concerns using POM
* Reusable utility methods
* Explicit waits for stability
* Environment-based execution (headless vs UI)
* Clean and maintainable code structure
